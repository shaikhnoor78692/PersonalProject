## 🚨 Warnings & Lessons Learned

### Facebook Scam Group:
```
"Mumbai Business Network" on Facebook is a SCAM
Red flags: Asks to WhatsApp admin before posting
           Promises 25 WhatsApp groups for free
           Posts get deleted if no WhatsApp
Action: Left the group immediately
```

### My Web Fetch Limitation:
```
Claude's web_fetch tool shows cached versions
Cannot confirm live website changes visually
Solution: User takes screenshots and shares
          Claude reviews screenshots directly
```

### LinkedIn DM Limitation:
```
LinkedIn DM requires Premium for non-connections
Free workaround: Comment on their posts instead
Comments visible to 200+ people = better reach
```
