# ScaleSnap — AI Services Agency

## 🚀 Project Overview
ScaleSnap is an AI-powered content writing and digital marketing agency website built to generate freelance income. The website offers content writing, social media, email marketing, video scripts, SEO strategy, and chatbot setup services.

## 🌐 Live Website
- **Website:** https://scalesnap.in
- **Hosted on:** Netlify (Free Plan)
- **Domain:** GoDaddy


## 🚀 Deployment
Site auto-deploys via GitHub → Netlify integration.

### To make changes:
```bash
# Edit files locally
# Then push to GitHub:
git add .
git commit -m "your change description"
git push
# Netlify auto-deploys in ~30 seconds
```

### Manual deploy:
- Go to app.netlify.com
- Click Deploys → Trigger deploy
